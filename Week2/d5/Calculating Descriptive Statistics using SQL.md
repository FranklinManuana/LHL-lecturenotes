## Calculating Descriptive Statistics

Instruction

Follow along with [this resource from SILOTA](http://www.silota.com/docs/recipes/sql-summary-statistics.html) to calculate descriptive statistics using SQL.

This [piece by Jean-Nicholas Hould](https://www.jeannicholashould.com/descriptive-statistics-in-sql.html) is another great resource that walks us through how to calculate descriptive statistics using SQL, but using the SQLite flavour of SQL. Although we have not yet explored that flavour, this resource does a great job of demonstrating how to gain insights using descriptive statistics and is worth reviewing, but not executing. In particular, the tutorial underlines how descriptive statistics can be leveraged to:

- quickly get a feel for the data
- investigate if there are outliers
- determine if the data is the shaped abnormally
