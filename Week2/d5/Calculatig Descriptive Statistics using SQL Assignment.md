## Calculating Descriptive Statistics

For this activity, we will continue using our Chinook database.

Instruction

Using the Invoice table of the Chinook database, complete the following tasks:

**Task 1**: Write a query to find out the total amount invoiced of all time.

**Task 2**: Write a query to find out how are the sales (invoice amounts) distributed over the years.

**Task 3**: Write a query to find out the average value of invoices by country and by year.

**Task 4**: Write a query to find out the number of unique customers by country.

**Task 5**: Write a query to find out the maximum and minimum invoice amount by country and by year.
