One of the advantages of a regular expression is that we can use the same syntax in all programming languages. Of course, we will be working with it in Python.

In this activity, we will learn the basic structure of regex and how to use it to match some text.

Instruction

Follow the interactive tutorial from [RegexOne](https://regexone.com/lesson/introduction_abcs). It will guide us, starting from the simplest patterns to the more complex ones.

You can also read through the PDF copy of this [Regex tutorial](https://drive.google.com/file/d/1zbpjNPCAwEXOdxdu-sL8haCe719YI5mE/view?usp=drive_link) from Medium instead.
