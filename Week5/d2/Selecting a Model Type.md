## Selecting a Model Type

### When to Apply a Regression Model

Regression algorithms are used to determine continuous values such as price, income, age, etc.

### When to Apply a Classification Model

Classification algorithms are used to forecast or classify distinct values such as True or False, Spam or Not Spam, etc.

### Examples

Read the scenarios below and think about which model would work best to solve the problem presented. Spend a few minutes thinking through the scenario first before you toggle to see the answer!

**Scenario**: You are working as a financial data analyst and are tasked with forecasting both the sales amount for next week as well as customer counts.

**Scenario**: You are working as a financial data analyst in a financial services firm. You mostly work with stock market data, and your job is to build a model that predicts whether the price will go up or down for the S&P 500 in each of the following three days in the future.
